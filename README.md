## 小从雨现在可以……

| Command                     | Description             | Example                |
| :-------------------------- | :---------------------- | :--------------------- |
| `/help`                     | 呃啊                    |                        |
| `/ping`                     | 歪？是小从雨吗？        |                        |
| `/roll <number=100>`        | 在 \[0, number\] 里摇点 |                        |
| `/roll [arg1] [arg2] <...>` | 专治选择困难            | `/roll 芳乃 从雨 茉子` |
| `/[...是不是/要不要...]`    | 回答型人工智障          | `/芳乃酱可不可爱？`    |
| `/[sleep/睡觉]`             | 该睡觉啦zzz             |                        |
