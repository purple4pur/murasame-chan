## 小丛雨现在可以……

| Command                                     | Description                | Example                     |
| :------------------------------------------ | :------------------------- | :-------------------------- |
| `/help`, `/帮助`                            | 呃啊                       |                             |
| `/ping`                                     | 歪？是小丛雨吗？           |                             |
| `/roll <number=100>`                        | 在 `[0, number]` 里摇点    | `/roll 20`                  |
| `/roll [arg1] [arg2] <...>`                 | 专治选择困难               | `/roll 芳乃 丛雨 茉子`      |
| `/[...是不是/要不要...]`, `/[...有没有...]` | 回答型人工智障             | `/芳乃酱可不可爱？`         |
| `/sleep`, `/睡觉`                           | 该睡觉啦zzz                |                             |
| `/contact`, `/feedback`, `/反馈` + `[内容]` | 有什么想对我的主人说的呢？ | `/contact 丛雨酱你不够可爱` |
